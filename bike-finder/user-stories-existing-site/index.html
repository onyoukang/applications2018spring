User Story - Lance

  • As Lance, I want to buy my bike from a well designed website, so that I would not get scammed, because I think poorly designed websites has higher chance of scamming me.
  • As Lance, I want to buy a better bike than what I have right now, so that I can win the upcoming race.
  • As Lance, I want to buy a bike made of carbon, so that I can go fast.
  • As Lance, I want to buy a newer model, so that my bike can be better than others'.
  • As Lance, I want to find a road bike, so that I can buy the right type of bike that I need.
  • As Lance, I want to sort the products by price from high to low, so that I can spend more money and buy a better bike.
  • As Lance, I want to read the description, so that I can learn more details of the bike.
  • As Lance, I want a bike with frame size 57, so that it would be a perfect fit for me.
  • As Lance, I want to see what other customers also viewed, so that I can compare different models.
  • As Lance, I want to put the models I like in the wishlist, so that I can compare the models that I like later.
  • As Lance, I want to put the bike that I finally chose into the cart, so that I can purchase it.
